## Features

- **Syntax Highlighting** for:
  - armips directives (`.org`, `.headersize`, etc.)
  - MIPS instructions (JAL, ADDU, NOP, etc.)
  - Registers (s0, s3, r0, a0, sp, etc.)
  - Hexadecimal numbers (0x8010AF14)
  - Comments (`//` single-line and `/* */` block comments)
  - Labels

## Installation

1. Copy the extension folder to your VSCode extensions directory:
   - **Windows**: `%USERPROFILE%\.vscode\extensions\`
   - **macOS/Linux**: `~/.vscode/extensions/`

2. Restart VSCode

3. Open any `.asm` or `.s` file to see syntax highlighting

## Supported File Extensions

- `.asm`
- `.s`
